Запуск приложения:
 - запустите redis-server в коммандной строке(Docker не подходит, проверено).
 - откройте проект в IDE. Запустите Main.class.
 - программа самостоятельно заполнит данные в базе данных.
 - по очереди будет показывать пользователей в консоли.
 - в случае, когда один из пользователей оплатил ViP контент - он будет показан вне очереди(главная очередь показа - непрерывный, бесконечный процесс). 
 - остановку программы произведите самостоятельно.
